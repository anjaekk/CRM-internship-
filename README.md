# 🖼 BToolTek-CRM(Internship)
B2tech 기업에서 한달간 인턴활동을 하며 구현한 CRM(고객 관계 관리) 사이트로 영업사원들의 일정관리 및 캘린더를 통해 일정확인이 가능합니다. 데이터 설계 및 기획부터 AWS를 이용한 배포까지 진행하였습니다.
